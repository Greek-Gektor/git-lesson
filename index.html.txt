<ul>
<li>git clone путь к репозиторию</li> указать путь к репозиторию

<li>git status</li> - статус локального репозитория

<li>git add путь к файлу </li> - добавить новый файл под контроль гита

<li>git commit -am "addet new file index.html.txt"</li> - зафиксировать изменения и дать им комментарий

<li>git push</li> - отправть все закомиченные изменения в репозиторий на сервер

<li>git pull</li> забрать свежие обновления

</ul>
